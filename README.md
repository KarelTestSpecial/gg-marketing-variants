Four versions of the gorilla marketing page with different background colors.

| File | Color Code | Color |

| **gorilla-1.html** | #08960d4b | Green/Amber |
| **gorilla-2.html** | #08963f4b | Green (darker) |

| **gorilla-4.html** | #1d1e10 | Brown |

## Viewing

You can view the variants by:

1. **GitHub Pages** (once enabled): Each file will be available at `https://kareltest special.github.io/gg-marketing-variants/<filename>`
2. **Raw HTML**: Click the "Raw" button on each file in GitHub to open it directly
3. **Download**: Clone this repo and open each HTML file locally in your browser

## Enable GitHub Pages

To make these available as live web pages:
1. Go to Settings → Pages
2. Select "Deploy from a branch"
3. Choose `main` branch and `/` (root) folder
4. Save

Then access files at: `https://kareltestspecial.github.io/gg-marketing-variants/gorilla-0-08960d4b.html` etc.
